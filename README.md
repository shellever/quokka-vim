# quokka-vim

Love vim, love quokka smile.


Installation
------

```bash
$ git clone git@github.com:shellever/quokka-vim.git
$ cd quokka-vim

# base vimrc
$ ./quokka-vim.sh -ib

# extended vimrc
$ ./quokka-vim.sh -ie
```


Uninstallation
-----

```bash
# base vimrc
$ ./quokka-vim.sh -ub

# extended vimrc
$ ./quokka-vim.sh -ue
```
